MATCH (p:Project)-[rs:REQUIRES_SKILL]->(s:Skill)
WHERE toLower(s.skill_name) CONTAINS 'ai'
   OR toLower(s.skill_name) CONTAINS 'python'
   OR toLower(s.skill_name) CONTAINS 'deep learning'
MATCH (e:Employee)-[hs:HAS_SKILL]->(s)
WITH p, e, sum(importance * proficiency_weight) AS fit_score,
     collect(DISTINCT s.skill_name) AS matched_skills
RETURN e.full_name, e.designation, fit_score, matched_skills
ORDER BY fit_score DESC LIMIT 25

How It Works

01

Upload Any Data

02

The LLM agent profiles your data, detects foreign keys, and infers a full graph schema automatically.

03
