2. SCOPE OF WORK

The Service Provider shall design, develop, and deploy a next-generation cloud-native
data platform for the Client. The scope includes:

2.1 Migration of legacy Oracle data warehouse to Google BigQuery
2.2 Implementation of real-time data pipelines using Apache Kafka
2.3 Development of a unified data catalog and governance layer
2.4 Building self-service analytics dashboards using Tableau
2.5 Data quality framework implementation and monitoring

5. SERVICE LEVEL AGREEMENTS

5.1 System Availability: The data platform shall maintain 99.9% uptime measured monthly.
    Penalty: 2% credit per 0.1% below target.

5.2 Data Freshness SLA: Real-time pipelines shall deliver data within 5 minutes of source change.
    Penalty: $5,000 per incident exceeding 15 minutes.

5.3 Incident Response Time: Critical incidents (P1) shall receive initial response within 30 minutes.
    P2 incidents within 2 hours. P3 within 8 business hours.

5.4 Data Quality Score: Minimum 98% accuracy on validated data sets, measured weekly.

6. KEY PERFORMANCE INDICATORS

6.1 Pipeline Throughput: Minimum 10,000 events per second sustained.
6.2 Query Performance: 95th percentile dashboard query response under 3 seconds.
6.3 Migration Accuracy: 100% data reconciliation between source and target systems.
6.4 User Adoption: 80% of target users actively using dashboards within 60 days of go-live.
